# 背景设定

这个废弃的精神病医院有一项超前的神经科学研究，能通过被称之为“恐惧仪”的装置具象化人脑的恐惧，并实体化这个怪物以服从宿主的控制，可是，这项研究引起了巨大的争议，最终被一部分科学家雪藏起来了。
你是一名赏金猎人，但在今晚，你并不唯一，你的任务是带出医院内所有的恐惧仪，并杀掉其他人，根据黑暗森林法则，其他人类都是你的敌人，现在是午夜十二点，开始吧。

# 思路

首先告知这个场馆里面有多少个恐惧仪（一般就是玩家数量），玩家找到仪器之后可以开始生成怪物。
为了确保可玩性，仪器是在玩家进入特定房间之后刷新出来的，所以每个人都能拿到仪器，集齐仪器的方法是亲手或者通过怪物杀掉对方。
